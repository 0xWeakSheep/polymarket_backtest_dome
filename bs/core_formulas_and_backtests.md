# 论文核心公式与回测提取

本文档只提炼这篇论文里最核心、最值得落地回测的部分，目标是回答四个问题：

1. 论文最核心的公式是什么
2. 哪些内容最值得回测
3. 回测时每个变量和结果应设什么边界
4. 应该画哪些图，图上要看什么结果

---

## 1. 论文最核心的三个公式

### 1.1 基础事件合约的理论价值公式

最基础的理论价值定义是：

`p_t = E^Q[ 1_{事件发生} | F_t ]`

含义：
- `p_t` 是时刻 `t` 的事件合约价格
- 因为到期支付只有 `0` 或 `1`
- 所以当前价格等于风险中性测度 `Q` 下，对未来支付的条件期望

这条公式决定了：
- 论文里的 `p_t` 既是价格
- 也是风险中性概率

这是全文最底层的定价定义。

### 1.2 log-odds 转化公式

论文不直接建模 `p_t`，而是先转成：

`x_t = ln( p_t / (1 - p_t) )`

反过来：

`p_t = 1 / (1 + e^(-x_t))`

含义：
- `p_t` 只能在 `(0, 1)` 内变化
- `x_t` 可以在整个实数轴上变化
- 所以更适合做扩散、跳跃扩散、波动率和相关性建模

### 1.3 最核心的模型公式：风险中性鞅约束下的漂移

论文真正的核心是：

`x_t` 的漂移不是自由设定的，而是由“`p_t` 在风险中性测度下必须是鞅”反推出来。

鞅约束写成：

`0 = S'(x)·μ(t,x) + 1/2 · S''(x)·σ_b^2(t,x) + ∫[ S(x+z) - S(x) - S'(x)·χ(z) ] ν_t(dz)`

其中：
- `S(x) = 1 / (1 + e^(-x))`
- `μ(t,x)` 是 `x_t` 漂移
- `σ_b(t,x)` 是信念波动率
- `ν_t(dz)` 是跳跃测度
- `χ(z)` 是小跳截断项

解出来以后：

`μ(t,x) = - { 1/2 · S''(x)·σ_b^2(t,x) + ∫[ S(x+z) - S(x) - S'(x)·χ(z) ] ν_t(dz) } / S'(x)`

这条公式是全文最关键的理论创新。

它的实际意义是：
- 你不能随便假设市场“平均往上走”或“平均往下走”
- 一旦要求 `p_t` 是公平价格，`x_t` 的漂移就自动被锁定
- 真正还能交易和对冲的，只剩下波动、跳跃、相关性、共跳这些风险因子

### 1.4 一般衍生品的定价公式

对任意 payoff `g(x_T)`，理论价格 `V(t,x)` 满足：

`∂V/∂t + μ(t,x)·∂V/∂x + 1/2·σ_b^2(t,x)·∂^2V/∂x^2 + ∫[ V(t,x+z) - V(t,x) - (∂V/∂x)·χ(z) ] ν_t(dz) = 0`

终端条件：

`V(T,x) = g(x)`

这条 PIDE 的意义是：
- 基础事件合约可以用它解释
- 方差类产品可以用它解释
- 首次到达类路径产品也可以用它解释

## 2. 最值得回测的内容

从论文主线看，最值得回测的不是“做多方向”，而是下面四类。

### 2.1 核心回测一：未来短窗 log-odds 已实现方差预测

这是论文实验最核心的对象。

预测目标：

`RV_x(t, h) = Σ (Δx)^2`

更接近论文写法时，可以理解为未来窗口 `[t, t+h]` 上的已实现 log-odds 方差。

你要验证的是：
- RN-logit-JD 模型预测的未来方差
- 是否优于简单基线模型

这是最应该优先做的回测。

### 2.2 核心回测二：跳跃检测与跳跃贡献分解

论文非常强调：
- 事件市场不是纯扩散
- 新闻会带来跳跃
- 方差预测的一部分来自跳跃项

所以第二个核心回测应该是：

把未来方差拆成：
- 扩散贡献
- 跳跃贡献

看在不同市场状态下：
- 哪部分贡献更大
- 模型是否能在预定新闻窗口前后正确抬升跳跃风险预测

### 2.3 核心回测三：多事件相关性与共跳

论文不是只做单事件。

它还强调：
- 事件间扩散相关性 `ρ_ij`
- 事件间共跳结构

所以第三个重点回测是：
- 两个或多个相关市场之间
- 联动是否由平时相关驱动
- 还是由同一新闻触发的共跳驱动

这部分最适合做“事件簇”市场，比如同一政治主题、同一宏观主题、同一项目生态主题。

### 2.4 核心回测四：边界附近的风险行为

论文反复强调概率边界 `0` 和 `1` 附近会出现病态问题。

所以必须单独回测：
- 当 `p_t` 接近 `0` 或 `1` 时
- 模型预测是否变差
- `x_t` 变换后是否更稳定

这部分非常关键，因为加密预测市场常有极端概率区间。

## 3. 回测时必须明确的变量边界

### 3.1 概率边界

因为

`x_t = ln( p_t / (1 - p_t) )`

所以当 `p_t = 0` 或 `p_t = 1` 时会发散。

因此实际回测必须裁剪概率：

`p_t ∈ [ε, 1 - ε]`

建议边界：
- 首选 `ε = 10^-5`
- 若数据噪音更大，可用 `ε = 10^-4`
- 不建议大于 `10^-3`，否则会明显扭曲接近结算时的极端概率

这条边界是硬条件，不做会直接导致 `log-odds` 爆炸。

### 3.2 时间窗口边界

论文偏向短期限预测，所以回测窗口 `h` 不要太长。

建议：
- 主窗口：`h = 60 秒`
- 补充窗口：`h = 300 秒`
- 长窗口上限：`h = 900 秒`

如果超过 `900 秒`，你回测的就不再是论文最核心的“短期做市/信念波动预测”问题了。

### 3.3 价格区域边界

为了分析不同状态，建议把 `p_t` 分成三段：

- 边界区：`p_t ∈ [ε, 0.1] ∪ [0.9, 1-ε]`
- 中间区：`p_t ∈ (0.1, 0.9)`
- 摇摆核心区：`p_t ∈ [0.25, 0.75]`

其中：
- 边界区用于测试数值稳定性和极端事件
- 中间区用于看一般状态
- 摇摆核心区最适合观察做市风险和信念波动

### 3.4 跳跃识别边界

如果你没有直接复现论文 EM 流程，至少要先设一个统一的“候选跳跃”阈值。

建议先用标准化增量：

`J_t = |Δx_t| / σ̂_local`

候选跳跃边界建议：
- 弱阈值：`J_t >= 3`
- 标准阈值：`J_t >= 4`
- 强阈值：`J_t >= 5`

画图时最好三档都出，这样才能看结论是否稳健。

### 3.5 相关性边界

两事件扩散相关性满足：

`ρ_ij ∈ [-1, 1]`

但实际回测时建议分桶：
- 强负相关：`[-1, -0.5)`
- 弱负相关：`[-0.5, 0)`
- 弱正相关：`(0, 0.5]`
- 强正相关：`(0.5, 1]`

这样方便看共跳与平时联动是否一致。

## 4. 回测时要拿到哪些结果

这一部分最重要。你不是只要“模型准不准”，而是要拿到能支持论文主张的结果。

### 4.1 结果一：未来方差预测误差

必须拿到：
- `MSE`
- `MAE`
- `QLIKE`
- 若有条件，可加 `logMSE`

输出边界：
- 全样本结果
- 边界区结果
- 中间区结果
- 摇摆核心区结果
- 预定新闻窗口结果
- 非预定时段结果

如果这些分组结果拿不到，你就无法判断模型到底在哪些市场状态下更有效。

### 4.2 结果二：方差贡献拆解

必须把预测方差拆成：
- 扩散项贡献
- 跳跃项贡献

至少要拿到下面这些比例：

`jump_share = 跳跃贡献 / 总预测方差`

建议输出：
- 全样本平均 `jump_share`
- 新闻窗口平均 `jump_share`
- 非新闻窗口平均 `jump_share`
- 边界区平均 `jump_share`
- 摇摆核心区平均 `jump_share`

这样才能验证论文关于“新闻跳跃对事件市场很关键”的主张。

### 4.3 结果三：相关性与共跳拆解

对于事件对 `(i, j)`，至少拿到：
- 扩散相关估计 `ρ_ij`
- 共跳频率
- 共跳贡献占总协方差比例

建议定义：

`cojump_share = 共跳协方差贡献 / 总协方差`

输出边界：
- 强相关事件对
- 弱相关事件对
- 同主题事件对
- 跨主题事件对

### 4.4 结果四：边界稳定性

必须拿到：
- 在 `p_t` 靠近 `0` 或 `1` 时，模型误差是否明显增大
- 用 `p` 直接建模与用 `x = log-odds` 建模相比，哪个更稳定

建议输出：
- 不同 `p` 区间下的 MSE 曲线
- 不同 `p` 区间下的 QLIKE 曲线

这会直接回答论文为什么坚持在 `logit/log-odds` 域工作。

## 6. 最低可执行回测方案

如果你只想先做一版最小可行验证，建议只做下面三件事。

### 6.1 第一优先级

做 `h = 60 秒` 的未来 log-odds 已实现方差预测：
- 目标变量：`RV_x(t, 60s)`
- 指标：`MSE`、`MAE`、`QLIKE`
- 分区：全样本、边界区、摇摆核心区

### 6.2 第二优先级

做跳跃贡献分解：
- 输出 `jump_share`
- 比较新闻窗口和非新闻窗口

### 6.3 第三优先级

做双市场相关性和共跳图：
- 选两到三个强相关市场
- 画 `Δx_i` 和 `Δx_j` 的联动
- 标记共跳时点

## 7. 最终你应该拿到的明确结论

做完以后，你至少应该能回答下面这些问题。

1. 用 `log-odds` 建模，是否比直接在概率 `p` 空间建模更稳定？
2. RN-logit-JD 是否在短窗未来方差预测上优于简单基线？
3. 事件市场的风险中，跳跃贡献占比有多大？
4. 在预定新闻窗口前后，跳跃风险和信念波动是否明显抬升？
5. 多个相关事件的联动，主要来自平时扩散相关，还是来自新闻触发的共跳？
6. 在 `p` 接近 `0` 和 `1` 的边界区，模型误差如何变化？
7. 能否得到一个稳定的 `σ_b(τ, x)` 或 `σ_b(τ, m)` 曲面？

如果这些问题都能拿到清晰结果，你的回测就已经抓住了这篇论文最核心的部分。
